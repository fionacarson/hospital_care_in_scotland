Hypothesis Test

Is summer is significantly different to winter?

The null hypothesis is that the average value for summer is the same or lower than the average value for winter.

H0 : mean winter - mean summer <= 0 HA : mean winter - mean summer >0

we are setting alpha for this test to be 0.05

alpha_test=0.05
smoothed_data %>% 
  filter(iswinter) %>% 
  ggplot() +
  aes(x = mvar) +
  geom_histogram(col = "white")

seasonal_avg <- smoothed_data  %>%
  group_by(iswinter) %>% 
  summarise(mean= mean(mvar, na.rm=TRUE))
seasonal_avg
null_distribution <- smoothed_data  %>%
  filter(iswinter) %>%  
  specify(response = mvar) %>%
  hypothesize(null = "point", mu = 6.04) %>%
  generate(reps = 1000, type = "bootstrap") %>%
  calculate(stat = "mean")
null_distribution %>%
  visualise(bins = 30) + 
shade_p_value(obs_stat = seasonal_avg$mean[seasonal_avg$iswinter]
            , direction = "less")

 
p_value <- null_distribution %>% 
get_p_value(obs_stat = seasonal_avg$mean[seasonal_avg$iswinter]
            , direction = "less")
